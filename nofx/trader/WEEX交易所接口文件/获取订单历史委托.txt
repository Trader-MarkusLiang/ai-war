获取订单历史委托
GET /capi/v2/order/history
权重(IP): 10, 权重(UID): 10

请求参数

参数名	类型	是否必填	说明
symbol	String	否	交易对
pageSize	Integer	否	每页数据的条数
createDate	Long	否	开始时间(天数必须小于等于90，不能为负数)
Unix毫秒时间戳
请求示例

curl "https://api-contract.weex.com/capi/v2/order/history?symbol=cmt_bchusdt&pageSize=10&createDate=1742213506548" \
   -H "ACCESS-KEY:*******" \
   -H "ACCESS-SIGN:*******" \
   -H "ACCESS-PASSPHRASE:*****" \
   -H "ACCESS-TIMESTAMP:1659076670000" \
   -H "locale:zh-CN" \
   -H "Content-Type: application/json"


返回参数

参数名	类型	说明
symbol	String	交易对
size	String	委托数量
client_oid	String	客户端标识
createTime	String	创建时间
Unix毫秒时间戳
filled_qty	String	成交数量
fee	String	手续费
order_id	String	订单id
price	String	委托价格
price_avg	String	成交均价
status	String	订单状态
pending：委托单已提交撮合，但未收到处理结果
open：委托单已被撮合引擎处理(已挂单)，可能部分成交
filled：委托单已完全成交【终态】
canceling：正在取消处理中
canceled：委托单已被取消。可能部分成交。【终态】
untriggered：条件委托单尚未被触发
type	String	委托类型
open_long：开多
open_short：开空
close_long：平多
close_short：平空
offset_liquidate_long：减仓平多
offset_liquidate_short：减仓平空
agreement_close_long：协议平多
agreement_close_short：协议平空
burst_liquidate_long：爆仓平多
burst_liquidate_short：爆仓平空
order_type	String	订单类型
normal：普通限价单，一直有效直至取消
postOnly：只做 maker 订单
fok：全部成交或立即取消
ioc：立即成交并取消剩余
totalProfits	String	总盈亏
contracts	Integer	订单张数
filledQtyContracts	Integer	已成交张数
presetTakeProfitPrice	String	预设的止盈价格
presetStopLossPrice	String	预设的止损价格
